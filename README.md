# postgresx

`postgresx` is the PostgreSQL runtime foundation library for `x.go`.

It provides the infrastructure layer around PostgreSQL: `pgxpool` lifecycle,
configuration, sqlc-compatible execution, explicit transactions, retry policy,
migration runner, error normalization, health checks, pool statistics,
observability adapters, secret masking, and integration test helpers.

It is not an ORM, does not own business schema, and does not depend on `x.go`.

## Install

```sh
go get github.com/bytechainx/postgresx
```

## Client

```go
client, err := postgresx.Open(ctx, postgresx.Config{
    DSN:             os.Getenv("POSTGRES_DSN"),
    ApplicationName: "market-data",
    MaxConns:        16,
})
if err != nil {
    return err
}
defer client.Close()
```

Applications that already build a `pgxpool.Config` can use `OpenPool(ctx, cfg)`.
That path preserves application-owned pgx hooks and still returns the
`postgresx.Client` lifecycle, health, stats, metrics, tracing, and sqlc API.
Metrics and tracing adapters emit only when `Config.EnableMetrics` and
`Config.EnableTracing` are enabled.

## sqlc

`postgresx.DBTX` matches the pgx interface generated by sqlc.

```go
q := generated.New(client.DB())
```

Inside a transaction:

```go
err := client.WithinTx(ctx, pgx.TxOptions{}, func(ctx context.Context, db postgresx.DBTX) error {
    q := generated.New(db)
    return q.UpdateCursor(ctx, params)
})
```

## Migrations

Applications own their migration files. `postgresx` only runs them.

```go
migrator, err := postgresx.NewMigrator(postgresx.MigrationConfig{
    DatabaseURL:    os.Getenv("POSTGRES_DSN"),
    MigrationsPath: "db/postgres/migrations",
})
if err != nil {
    return err
}
defer migrator.Close()

return migrator.Up(ctx)
```

`MigrationConfig.SourceURL` can be supplied directly for non-default migrate
sources. `MigrationsPath` is converted to a `file://` source URL, and
`MigrationsTable` is passed through to the pgx migrate driver. The migrator also
supports `Down(ctx, steps)` and `Version(ctx)`.

## Secret Masking

Use `postgresx.MaskDSN` before logging configuration or connection failures.

```go
log.Printf("postgres dsn=%s", postgresx.MaskDSN(os.Getenv("POSTGRES_DSN")))
```

## Verification

```sh
make ci
POSTGRES_TEST_DSN="$POSTGRES_TEST_DSN" make integration
```

`POSTGRES_TEST_DSN` is required only for live PostgreSQL integration tests.
