## Release caveats

This evidence is sufficient for a local convergence snapshot only. Before a
published release, add fresh CI logs, tag and remote publication evidence,
checksum artifacts, and contract hashes. Before a downstream adoption claim,
add current consumer-side dependency, compile, test, import-boundary, and
release-manifest evidence.
