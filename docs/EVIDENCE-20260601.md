## Canonical Commands

```sh
GOWORK=off go list -m
GOWORK=off go list ./pkg/postgresx
GOWORK=off make ci
GOWORK=off make integration
GOWORK=off make release-evidence-check
GOWORK=off make evidence
```

Live PostgreSQL integration is optional for ordinary local checks when neither
`POSTGRESX_INTEGRATION_DSN` nor `POSTGRES_TEST_DSN` is set and Docker is not
available. Release preflight can make it mandatory with
`POSTGRESX_REQUIRE_INTEGRATION=1`.

## Remaining Release Gaps

- Remote tag publication and GitHub release upload are outside this local worker
  task.
- Live PostgreSQL evidence depends on a DSN or Docker availability in the runner.
